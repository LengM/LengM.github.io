基本类型、引用类型、String

基本类型的特征

不同的变量会分配不同的存储空间，并且存储空间中存储的是该变量的值。赋值操作传递的是变量的值，改变一个变量的值不会影响另一个变量的值

引用类型的特征

赋值是把原对象的引用传递给另外一个引用。对数组而言，当一个数组引用赋值给另外一个数组引用后，这两个引用指向同一个数组，也就是指向同一块内存空间。

我们自己定义的实体类，实际上就是一种引用数据类型

String类型

我们一般都会说String是不可变的，那么什么是不可变的呢？

  • String是一种特殊的引用数据类型，它的特征跟基本类型基本一致
  • 对于String来说，不可变不意味着不能赋值
  • 由于String有很多方法，“不可变”属性保证了你的字符串在调用方法之后，不会改变原字符串的内容
  1. 创建引用s指向值null
  2. 进入stringParam方法，此时参数传入的实际上是引用s的值null，进入方法后相当于又创建了引用str指向值null
  3. 将引用str赋值，此时的赋值与引用s毫无关系
  1. 创建引用strings指向数组对象
  2. 进入stringArrayParam方法，此时通过参数传入的是引用strings，进入方法后，引用strArr和引用strings共同指向同一个值
  3. 对strArr进行操作实际上就是对strings进行操作
